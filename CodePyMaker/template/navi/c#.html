Variables

Crear variables

Las variables son contenedores para almacenar datos. En C# las variables tienen la pecularidad que deben ser declaras y posteriormente iniciadas, esto se puede realizar en la misma linea, como tambien varias mas abajo.

La estructura que que se debe seguir al declacar un variable es:

  • tipoDeDato nombreVariable = el valor que se le quiera entregar

Ejemplo:

int x = 4
string y = 'hola'

Nombrar una variable

Una variable puede tener un nombre corto (como x o y) o uno más descriptivo (como edad,fecha,area_total).Sin embargo C# se tienen en cuenta estas suguerencias:

  • No comenzar el nombre de una variable con guion bajo "_"

  • No crear mas de una variable que se diferencie solo por una letra

    ejemplo
    • edad
    • Edad
    • EDAD

  • Comenzar el nombre de las variables con minuscula

  • Si el nombre de una variable esta compuesto por mas de una palabra, comienza la segunda con mayúscula (camelCase)

Salidas de variables

En C# el comandando utilisado para enviar mensajes por panralla desde la consola (CMD) es "Console.WriteLine()".

Ejemplo:

string x = 'hola mundo'
Console.WriteLine(x)

En la salida:

hola mundo

Console.WriteLine() acepta cualquier tipo de dato y lo transforma a un texto para despues mostrarlo

Condicionales

Condiciones:

todos los controladores de flujo dentro de lo que hacen deben estar dentro de estas llaves "{}"

Estos son los que acompañarán a las declaraciones "if" o "elif" y bucles.

Las condiciones de los if tanto como los else if deben ir entre pararentesis siempre

Igual: a == b
No es igual: a != B
Menos de: a < b
Menor o igual que: a <= b
Mayor que: a > b
Mayor o igual que: a >= b

Declaración if:

Se escribe una "declaración if" utilizando la palabra clave reservada if.

Ejemplo:

int a = 50;
int b = 100;
if (b > a):
{
Console.WriteLine("b es mayor que a");
}

En este ejemplo, usamos dos variables, a y b, que se usan como parte de la instrucción if para probar si b es mayor que a. Como a es 50 y b es 100, sabemos que 100 es mayor que 50, por lo que imprimimos en la pantalla que "b es mayor que a".


Else if

La palabra clave else if es la forma de c# por si la codicion del if es falsa, entonces prueba con esta condición.

Ejemplo:

int a = 50;
int b = 50;
if (b > a):
{
 Console.WriteLine("b es mayor que a");
}
else if (b == a):
{
 Console.WriteLine("b es igual que a");
}

En este ejemplo, a es igual a b, por lo que la primera condición no es verdadera, pero la condición else if es verdadera, por lo que se imprime en la pantalla que "a y b son iguales".


Else

La palabra clave else captura cualquier cosa que no esté atrapada por las condiciones anteriores.

Ejemplo:

int a = 100;
int b = 50;
if (b > a):
{
 Console.WriteLine("b es mayor que a");
}
else if (b == a):
{
 Console.WriteLine("b es igual a");
}
else:
{
  Console.WriteLine("a es mayor que b");
}

En este ejemplo, a es mayor que b, por lo que la primera condición no es verdadera, también la condición else if no es verdadera, así que pasamos a la condición else y mostramos en la pantalla que "a es mayor que b".


También puedes tener un else sin el else if:

Ejemplo:

int a = 100;
int b = 50;
if (b > a):
{
 Console.WriteLine("b es mayor que a");
}
else:
{
  Console.WriteLine("a es mayor que b");
}

Y Logico

El simbolo del y Logico es "&&", es un operador lógico, y se utiliza para combinar declaraciones condicionales

Ejemplo:

int a = 100;
int b = 50;
int c = 150;
if (a > b && c > a):
{
  Console.WriteLine("c es mayor que b")
}

En este ejemplo se comprueban dos condiciones y si ambas se cumplen muestra que "c es mayor que b".


O Logico

El simbolo clave es "||" es un operador lógico y se usa cuando basta que una de las dos condiciones sea verdadera

Es la tecla al lado izquierdo del 1

Ejemplo:

int a = 100;
int b = 50;
int c = 150;
if (c > b || c > a):
{
  Console.WriteLine("C");
}

En este ejemplo se comprueban 2 condiciones y si una de las dos se cumple se imprime "C".


Anidado de condicionales

Puede tener declaraciones if dentro de declaraciones if, esto se llama instrucciones if anidadas.

Ejemplo:

if (a > b):
{
 if (c > b):
   {
   Console.WriteLine("O");
  }
}

En este ejemplo se comprueba la primera condición "a > b", si esta se cumple, compruba si "c > b", y esi esta última se llega a cumplir imprime "O".



Ciclo while

Con el ciclo while podemos ejecutar un conjunto de declaraciones siempre que una condición sea verdadera, se debe considerar crear un cierre a esta para que no se repita de manera indefinida.

Al igual que con los condicinales la condicion del While debe de estar entre parentesis

Ejemplo:

int i = 1
while (i < 6):
{
 Console.WriteLine(i);
 i += 1;
}

El tiempo del bucle depende de la condición, como en este caso la variable "i" va aumentando de 1 en 1 hasta llegar a 6 y así "romper el ciclo" para salir de este.


Funciones/Metodos

Para que un metodo pueda ser ejecutado debe ser llamado desde el main, de lo contrario no se ejecutara y nunca realizara nada.

using System;
namespaces CodePyMaker {
 class program{
  static void Main(String[] args){
   
  }
 }

consideraciones:

  • No es ejecutado hasta que es llamado en el Main.
  • Todos los metodos deben estar dentro de una Clase.
  • Se debe especificar el tipo devuelto y los parametros.
  • No existe diferencia entre metodos y funciones en C#, ambos terminos son lo mismo.
  • Siempre los metodos deben llevar un return, a exepcion los que sean del tipo void.

Son un grupo de sentencias a las que se les da un nombre identificado, que realiza una tarea en concreto

Se le pueden pasar datos conocidos como parametros, tambien no se le pueden pasar parametros siempre y cuendo esta este diseñada para funcionar asi.

Un metodo puede devolver valores como no puede hacerlo esto depende de su contruccion.


Creacion de un metodo

En C# un metodo se define siempre usando la palabra reservada "static" bajo la siguiente sintaxis:

static tipoDato + nombreMetodo (parametros){

 cuerpo del metodo

}

Si no utilizamos la palabra reservada "static" se generara un problema de objeto, de momento es recomendable usarla solamente aunque no sepamos que significa. En una proxima actulizacion se agregata POO(Programacion Orientada a Objetos).

nota: tipoDato puede ser int, double, string o cualquier tipo mencionado en la seccion de tipos de datos.

Ejemplo de un metodo que retorna un valor:

static void Suma(){:
 int numero = 7;
 int numero1 = 9;
 int resultado = nummero + numero1;
 Console.WriteLine(resultado);
}

nota importante: los metodos con la palabra reservada "void" nunca llevan return

utilizamos la palabra reservada "void" para indicar que no retornaremos nada

static int Suma(int num1, int num2){:
 int resultado = num1 + num2;
 Console.WriteLine(resultado);
}

nota importante: todos los metodos que sean de tipo numerico, string o arrays simpre deben llevar un return

Llamado de un metodo

Para llamar a un metodo este debe estar dentro del Main.

Ejemplo:

using System;
namespaces CodePyMaker {

  static void Main(String[] args){
   metodo();
  }
 }
}
static void metodo(){
 Console.WriteLine("Hola funcion");

Parámetros

La información puede suministrarse al metodo como parámetro.

Los parámetros se especifican después de que el nombre del metodo, dentro de los paréntesis "()". Puede añadir tantos parámetros como desee, sólo les separan con una coma.

El siguiente ejemplo tiene un metodo con un parámetro (variable). Cuando se invoca el metodo, se pasa a lo largo de un nombre de pila, que se utiliza dentro de la función para imprimir el nombre completo

Al pasarle parametros a un metodo siempre se deben definir de que tipo son como se muestra a continuacion.

using System;
namespaces CodePyMaker {
 class program{
  static void Main(String[] args){
   metodoConParametros("este es mi");
   metodoConParametros("hello");
   metodoConParametros("CodePyMaker creo un");
  }
 }
}
static string metodoConParametros (string variable)
{
 string fusion = variable + " metodo";

 return fusion;
}

Se puede enviar cualquier tipo de datos de parámetro a un metodo (cadena de string, número, arrays,, etc.), y será tratado como el mismo tipo de datos dentro del metodo, siempre y cuando coincida con el tipo de dato en las variables del metodo.


Parámetros por defecto

Si llamamos al metodo sin parámetros, se utiliza el valor por defecto.

Ejemplo:

using System;
namespaces CodePyMaker {
 class program{
  static void Main(String[] args){
   Console.WriteLine(metodo());
   Console.WriteLine(metodo("nuevo"));
  }
 }
}
static string metodo(string varibles="CodePyMaker")
{
 string fusion = variable + " " + " metodo";

 return fusion;
}
//la salida de la primera linea es CodePyMaker
// la segunda con el str "nuevo" es "nuevo metodo"

Como se ve en el ejemplo para hacer un parámetro por defecto se debe asignar un valor a la variable de la función, que en caso de ser llamada sin parámtros automáticamente tomará ese. Siempre se deba tener encuanta que estas siempre van al final de los parametros obligatorios, no se pueden intercalar.

Ciclo for

Un ciclo for se utiliza para iterar sobre una secuencia (que es ya sea una lista, una tupla, un diccionario, un conjunto, o una cadena).

Existen dos formas de for en c# uno que esta dedicado especificamen a iterar sobre datos mutables

sintaxis:

for (iniciacio del bucle, condicion, incremento/dremento)
{
 codigo
}
foreach( iterador in nombre)
{
 codigo
}

Ambos son aplicable para cualquier tipo de dato pero es mas recomendable utlizar el foreach en la iteracion de algun tipo de dato


ejemplos

using System;
namespaces CodePyMaker{
 class program {
  static void Main(String[] args) {
   impar = 0;
    for (i = 0; i < 100; i++)
     {
      if (i % 2 != 0)
       {
        Console.Write("{0}|", i);
        impar++;
       }
     }
   }
  }
}


Strings

Se define un string por el o los caracteres rodeados por comillas simples o dobles ("" o '') siendo "hola" lo mismo que 'hola'.

Ejemplo:

Console.WriteLine("Hola")
Console.WriteLine('Hola')

En ambos casos se imprime lo mismo (Hola).


Operaciones con string

Los estring se pueden asignar a las variables y operar en ellas, con sumas de strings, multiplicación de un string por un número entero. Recordar que nunca se puede sumar un string con un valor numérico a menos que sea de la forma string + string(valor)

Ejemplo:

string palabra = "Hola"
string otra_palabra = "Mundo"
Console.WriteLine(palabra + " " + otra_palabra)
Console.WriteLine(palabra * 2)

En el caso de la suma lo que resultaría sería Console.WriteLine("Hola Mundo") y en la multiplicación Console.WriteLine("HolaHola")


Índices de un string

Un string funciona como una matriz, por lo cual tiene índice, siempre partiendo desde el índice 0.

string pa = "Hola"
Console.WriteLine(pa[0])

En este caso de la variable pa se muestra "h"


Verificación

Se puede comprobar si un caracter o una palabra está en otro string utilizando la función in, y en contraparte a esta se puede usar el not in de modo que si la respuesta fuera true en el caso anterior en este sería false.

Ejemplo:

string texto = "Hola como estás"
string p = "como"
if (p in texto){
 Console.WriteLine("Si")
}
else if (p not in texto){
}
 Console.WriteLine("No")
else{
 Console.WriteLine("Si")
}

Como en ambos casos existe p en texto de igual forma imprimirá Si

Tuplas

Las tuplas son datos que no son modificables y no pueden ser ordenadas. y son escritas con '()'

Crear una tupla

Para crear una tupla nueva se utiliza tuple(), en cambio con valores, se colocan los valores dentro de ()

Ejemplo:

var tupla = (valor1, valor2);
var tupla2 = ('gato','perro','loro');
una tupla puede recibir hasta n valores.

Acceder a las tuplas

Para acceder al valor de una tupla se debe hacer con la palabra reservada item mas la posicion del elemento

Ejemplo:

var y = ('francia','españa','china');
Console.WriteLine(y.items1);

En la salida:

francia

Ciclo atraves de la tupla

Estos pueden ser recorridos por un ciclo for

Ejemplo:

var frutas = ('manzana','pera','uva', 'melon')
for (i in frutas)
{
 Console.WriteLine(i);
}

En la salida:

manzana
pera
uva
melon
